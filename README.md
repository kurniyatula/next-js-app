## Let's code!

- Clone this repository by running `git clone https://github.com/kurniyatula/next-js-app`

```bash
npm install
```

#### For development

```bash
npm run dev
```

#### For production

```bash
npm run build
# and 
npm run start
```

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
